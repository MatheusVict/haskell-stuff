# Haskell

### Basics
**Arrays operations**


```haskell
head [1,4,5,6] -- Get the first element
```
> output: 1

```haskell
tail [1,2,3,4,5] -- Remove the first element from a list
```
> output: [2,3,4,5]

```haskell
[1,2,3,4,5] !! 2 -- Select the nth element of a list
```
> output: 3

```haskell
take 3 [1,2,3,4,5] -- Select the first n elements of a list
```
> output: [1,2,3]
